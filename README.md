# IsNaiveBayesReliable
朴素贝叶斯分类器是文本情感分析中最基础的机器学习算法之一，其核心假设是特征之间的条件独立性。然而，在实际文本分类任务中，这个假设往往并非完全成立——词语之间存在着一定的共现关系。本文以IMDB电影评论的情感分类为研究对象，从条件概率和协方差的角度量化分析独立性假设的失效程度，并通过对比实验验证：独立性假设在电影评论领域失效程度极低（相关系数>0.3的词对占比不足万分之一），且这种失效对朴素贝叶斯分类器的性能影响较小（三种特征方法准确率差异<1%）。此外，本文还通过不同样本规模下的条件概率估计，验证了大数定律在统计学习中的收敛特性——当训练样本从100增加到12500时，条件概率估计的标准差减小了两个数量级，表明估计趋于稳定可靠。
